# 🚀 uuid-lab

> The most advanced ID generation toolkit ever built for Node.js & JavaScript  
> ⚡ 200+ utilities • 🔐 Secure • 🌍 Distributed • 🧠 Smart • 📦 Production-ready

## Installation
npm install uuid-lab

## Quick Start
const uid = require('uuid-lab');

uid.nanoId();
uid.uuid();
uid.ulid();
uid.snowflakeId();

## Features
- UUID, NanoID, ULID, KSUID
- Semantic IDs
- Lifecycle IDs
- Fuzzy IDs
- Compound IDs
- Hierarchical IDs
- Collision Detection
- Federation
- Compliance (GDPR)
- CLI Support

## CLI
npx uuid-lab generate nanoid
